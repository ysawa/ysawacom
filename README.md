# ysawa.com

ysawa.com is the site of Yoshihiro Sawa.

## Libraries and Frameworks

The application uses those libraries and frameworks.

* Ruby on Rails 3.2.13
* MongoDB
* jQuery 1.9.1
* Twitter Bootstrap
* Backbone.js
* Marionette.js
* HTML5/CSS3
